# Agent Lab

Experiments with LLM agents, tool use, RAG, and workflow automation.

This repository is planned as a lightweight lab for collecting agentic AI experiments, reusable prompts, tool wrappers, and small demos.

## Focus

- LLM agents and tool use
- Retrieval-augmented generation
- Prompt engineering and workflow design
- Coding assistants and research assistants
- Evaluation notes for agent behavior

## Repository Structure

```text
agent-lab/
├── README.md
├── requirements.txt
├── .env.example
├── agents/
├── docs/
├── examples/
├── prompts/
├── tools/
└── tests/
```

## Safety

Do not commit API keys, tokens, passwords, private documents, or private datasets.

Use `.env.example` to document required environment variables, and keep the real `.env` file local.
